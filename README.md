# Customer Satisfaction Prediction

## Project Overview
This project analyzes the relationship between customer satisfaction and repeat purchases using Ordinary Least Squares (OLS) Linear Regression. The objective is to understand how customer satisfaction influences customer loyalty and purchasing behavior.

## Dataset
The dataset contains two variables:

- Satisfaction: Customer satisfaction score
- Repeat_Purchases: Number of repeat purchases made by customers

Dataset Size:
- 99 Records
- 2 Features

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- Linear Regression (OLS)

## Project Workflow
1. Data Collection
2. Data Loading and Exploration
3. Descriptive Statistical Analysis
4. Data Visualization
5. OLS Linear Regression Model Building
6. Model Interpretation
7. Business Insights and Conclusions

## Key Concepts Covered
- Simple Linear Regression
- Ordinary Least Squares (OLS)
- Loss Function
- Slope and Intercept Calculation
- Statistical Modeling
- Customer Behavior Analysis

## Results
The analysis demonstrates a positive relationship between customer satisfaction and repeat purchases, indicating that higher customer satisfaction contributes to increased customer loyalty and repeat buying behavior.
